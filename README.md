# SpaceVision AI

SpaceVision AI is an artificial intelligence-based wildfire detection system.

## Project Overview

SpaceVision AI analyzes images using deep learning to detect potential wildfire presence and classify risk levels.

The system calculates fire probability and automatically determines the level of risk.

## Features

- Fire / No Fire classification
- Probability output
- Risk level assessment
- Web-based interface
- Real-time inference

## Technology Stack

- Python
- Streamlit
- PyTorch
- MobileNetV2
- Plotly

## Installation

1. Install requirements:

pip install -r requirements.txt

2. Run the application:

streamlit run app.py
